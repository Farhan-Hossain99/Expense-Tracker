# FinPulse - Expense Tracker

A modern expense tracking web app built with Flask + SQLite backend and vanilla HTML/CSS/JS frontend.

## Features
- Dashboard with financial overview cards
- Spending breakdown with category chart
- Savings goals tracking
- Transaction history
- Responsive glassmorphism UI design

## Tech Stack
- **Frontend**: Vanilla HTML/CSS/JS, Tailwind CDN, Plus Jakarta Sans
- **Backend**: Flask (Python), Flask-CORS
- **Database**: SQLite

## Quick Start

### Backend
```bash
pip install -r requirements.txt
python app.py
```
Server runs at http://localhost:5000

### Frontend Only (no backend)
```bash
python -m http.server 3000
```
Open http://localhost:3000

## Project Structure
- `index.html` — Main dashboard
- `style.css` — Custom styles (glass cards, vibrant colors)
- `script.js` — Tailwind config
- `app.py` — Flask API backend
